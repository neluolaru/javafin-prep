# javafin-prep

Java fundamentals prep for JavaFin/Switchfully.

## Requirements
- JDK 17+
- Maven 3.9+

## Structure
- `src/main/java/com/ionolaru/javafinprep/exercises` — 1 file per exercise

## Run
Build:
mvn -q -DskipTests package

Run an exercise (example):
java -cp target/classes com.ionolaru.javafinprep.exercises.Ex01_VariablesAndPrinting
